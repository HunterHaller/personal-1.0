Notable School Projects

  • Steadfast Station (Spring 2020)

    A game designed and built completely for the web! This was my capstone project to finish off my degree.

    Take on the role of a space station technician and tackle problems in orbit. The game simulates the operating system that the character would use, transforming your own desktop with draggable/closeable windows, icons, a command terminal, and a chat window.

    Visually built on HTML and CSS, mechanically built on JavaScript and JQuery.

    Utilizes the JQuery Terminal plugin.

    GitHub repo

    Playable here! (Best played in fullscreen on a desktop. Not built for mobile devices!)
